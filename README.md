# Aware

Aware is a menubar app for macOS and visionOS that displays how long you've been actively using your computer.

## Development information

Requires Xcode 10.2

``` sh
$ git clone https://github.com/josh/Aware
$ cd Aware/
$ open Aware.xcodeproj/
```
